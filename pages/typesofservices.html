Информационный центр «Максимус»

Налоговые услуги

Информационный центр «Максимус» оказывает следующие виды налоговых услуг:

  • Консультации по вопросам налогообложения;
  • Услуги сервисного центра по подключению индивидуальных предпринимателей и организаций к программному комплексу СКБ Контур;
  • Услуги по оформлению усиленной квалифицированной электронной подписи для сдачи отчетности в ФСС, ФСРАР, Центробанк;
  • Услуги по оформлению усиленной квалифицированной электронной подписи для торгов;
  • Услуги по оформлению электронной подписи для физлиц для получения финансовых услуг, для работы на государственных электронных порталах, подачи заявления на поступление в ВУЗ и получения электронных услуг через портал госуслуг: получение загранпаспорта, запись к врачу, постановка автомобиля на учет и т д.;
  • Бухгалтерское сопровождение организаций и индивидуальных предпринимателей;
  • Регистрация, внесение изменений в действующие документы, ликвидация организаций и индивидуальных предпринимателей;
  • Оказание услуг по составлению и передаче отчетности по телекоммуникационным каналам связи в ИФНС, ПФР и ФСС, РОССТАТ, ФСРАР и Комитет по регулированию оборотов алкогольной продукции, Росприроднадзор по Ставропольскому краю;
  • Оказание услуг по сдаче индивидуальных сведений в ПФР;
  • Помощь в составление физическим лицам отчетов по форме 3 НДФЛ;
  • Услуги по оплате взносов, налогов и штрафов;
  • Получение кодов статистики.

Для индивидуальных предпринимателей

  • оказание услуг по регистрации физического лица в качестве индивидуального предпринимателя на основании проведенной предварительно с ним беседы и проведению консультации по поводу его дальнейшей предпринимательской деятельности;
  • оказание услуг по постановке на учет по месту жительства /осуществления деятельности индивидуальных предпринимателей;
  • оказание услуг по внесению изменений в Сведения, содержащиеся в Едином государственном реестре индивидуальных предпринимателей;
  • оказание услуг по прекращению предпринимательской деятельности

Для организаций

  • оказание услуг по регистрации и ликвидации организаций;
  • оказание услуг по внесению изменений в Сведения, содержащиеся в Едином государственном реестре юридических лиц;

Прочее

  • составление заявлений различного характера;
  • передача и забор документов из ИФНС, ПФР;
  • получение актов сверок, справок по состоянию расчетов по налогам, сборам пеням и пр.;
  • получение выписок из ЕГР, копий Уставов.
